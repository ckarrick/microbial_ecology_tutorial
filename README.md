# microbial_ecology_tutorial
A basic introduction to analyzing microbiome data. \

microbial_ecology_tutorial.Rmd : R code to compute, statistically compare, and visualize alpha diversity, beta diversity, taxonomic composition, and differentially abundant microbes. \

data/ : Example dataset of 16S sequencing data from a meta-analysis of studies exploring bacterial communities in corals affected by stony coral tissue loss disease (Rosales et al., 2023)
